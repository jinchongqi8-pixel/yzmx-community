- ✅ **无需短信验证码**

### 测试步骤
1. 打开登录页面
2. 输入任意手机号（如：13800138000）
3. 输入任意6位验证码（如：123456）

5. 自动跳转到首页

## 🚀 如何切换到生产模式

### 第一步：配置短信服务
1. 查看 [阿里云短信配置指南](./ALIYUN_SMS_GUIDE.md)
2. 申请签名和模板
3. 获取 AccessKey
4. 修改云函数配置

### 第二步：修改代码

**1. 开启路由守卫**
打开 `src/router/index.js`，取消注释第 40-45 行

**2. 使用真实 API**
打开 `src/api/cloud.js`：
- 取消注释 `sendSmsCode` 中的真实云函数调用
- 注释掉开发模式的模拟代码
- 对 `phoneLogin` 同样操作

### 第三步：部署云函数
```bash
# 在微信开发者工具中
右键 cloudfunctions/login → 上传并部署：云端安装依赖
```

## 💡 开发技巧

### 使用浏览器控制台
```javascript
// 手动设置登录状态
localStorage.setItem('userInfo', JSON.stringify({
  nickname: '测试用户',
  coins: 100
}))

// 清除登录状态
localStorage.clear()

// 查看存储的数据
console.log(localStorage.getItem('userInfo'))
```

### 跳过登录测试功能
在 `src/router/index.js` 中，页面有 `meta: { requiresAuth: true }` 的需要登录
注释掉路由守卫后，所有页面都可以直接访问

## ⚠️ 注意事项

1. **开发模式仅用于测试**
   - 生产环境必须使用真实验证码
   - 开发模式的模拟数据不会被保存

2. **数据不会持久化**
   - 用户信息存储在浏览器
   - 清除缓存会丢失数据
   - 建议后续对接真实数据库

3. **云函数需要配置**
   - 云开发 HTTP API 需要开通
   - 需要配置环境 ID
   - 参考小程序配置
